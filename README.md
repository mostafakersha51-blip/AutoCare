# 🚗 AutoCare - Vehicle Service Platform

AutoCare is a modern, responsive web app for car service providers. It combines service booking with an e-commerce shop for car parts and accessories.

![React](https://img.shields.io/badge/React-19-blue.svg)
![TailwindCSS](https://img.shields.io/badge/Tailwind-3.4-cyan.svg)
![React Router](https://img.shields.io/badge/React_Router-7-red.svg)

## ✨ Features

- 📅 Appointment booking with date/time selection  
- 🛒 E-commerce shop with search, categories, and cart  
- 🛠 Service pages for repairs, towing, emergency help  
- 📍 Interactive service center maps with Leaflet  
- 📱 Fully responsive, mobile-first design  
- 🧠 State managed via React Context API  

## 📂 Project Structure

```text
src/
├── components/    # UI components, Navbar, Footer
├── context/       # Cart & Booking state
├── pages/         # Home, Shop, Services, Checkout
└── App.js         # Routing and context providers
```

## 🛠 Tech Stack

React 19 | Tailwind CSS | React Router v7 | Leaflet | React Icons | Poppins Fonts
